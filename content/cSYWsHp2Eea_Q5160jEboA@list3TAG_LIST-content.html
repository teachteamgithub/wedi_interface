Table @list:TAG_LIST

DescriptionThis table is a standalone table that holds tags that can be associated with a note. This table is used to populate screen with available tags.

This table would be configured for each district during the install. There are 7 predefined tags that would be implemented if the district did not decide to extend the list. The prefined values are listed in the TAG_TXT column definition.

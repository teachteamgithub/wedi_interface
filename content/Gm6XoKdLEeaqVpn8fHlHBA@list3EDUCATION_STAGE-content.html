Table @list:EDUCATION_STAGE

DescriptionThe grade level defined by a country and possibly state/province.

The district should be consulted as to what names are used when referring to grades. If the disctrict does not have a source for grade level, this file would be created as a flat file and maintained by data collection team. Once the initial creation is done not a lot of change is expected.

Reference article on values used around the world for grade level: https://en.wikipedia.org/wiki/Educational_stage.

Sample:
ISO_3166_2_COUNTRY_CD: CA-ON
EDUCATION_STAGE: Grade 8
SCHOOL_STAGE: Elementary

Column Details

 ISO_3166_2_COUNTRY_CD
DescriptionThe 5 character ISO code that identifies the major subsivisions within a country. . This code had to be used for countries like Canada and Austrailia where the grade levels differ between provinces/states. In the case where the grade level is named consistently across geographies, the province/state of the district should be used for the second portion of the code.

For example, Ontario which is a province in Canada is 'CA-ON' and California which is a state in the United States is 'US-CA'

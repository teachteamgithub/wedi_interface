Table @list:SIH_INTEREST_CATEGORY

DescriptionThere is a JSON file that contains the categories and url references. That should be used to load the table.

There is a predefined set of interests that are loaded when the application is configured.

Note from Austin Burrows on this data (5/24)
Yeah, interest categories are predetermined, but more categories could be added if needed. There is no mechanism in the app to add new categories. That would need to be a back end change. Teachers can't add new categories. They can only add comments to the given categories for a student. When the app is first used, a student profile has a list of 10 interest categories with 0 comments for each one.

Table MASTERY_COLOR_LIST

DescriptionStaging source / Algorithim input
This is a system table that hold configurable parameters that drive how mastery is visualized. This table is configured by the district and is used by all schools in the district. Up to 5 preset color ranges can be configured.

If a district decides it only wants 3 ranges, the expectation is the forecasting model would only deliver results in 3 bins.

The starting values for a district are
At Risk
Low Performing
Satisfactory
Mastered
Above Mastery
Needs Assessment
CreatedR1.2 - Enlight #575
Configurable Tableyes - mastery forecast result range needs to be defined for up to 5 colors

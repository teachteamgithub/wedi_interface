Schema InsightDataCollection

DescriptionDocuments the data to be collected from schools and school districts when populating Insight data repositories. This includes file layouts and list of things to collect and validate with the school district.

The file layouts have been synchronized with the file ALS Input Tables Required from School Districts_S2T(20160413).xlsx

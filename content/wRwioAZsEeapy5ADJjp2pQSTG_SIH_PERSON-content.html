Column Details

 PERSON_ID
DescriptionUnique identifier for a person being stored in Insight.

- For a teacher, principal, administrator (school district employee) this might be their employee number. We add 'T-' as a prefix to ensure no person id duplication.
- For a student this might be their student number. We add 'S-' as a prefix to ensure no person id duplication.
- For a contact if there is no source system id this can be fabricated using this pattern 'student_id + contact lastname + contact first name + relationship'
